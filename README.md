## Architecture / UI principles
This repository is organized around reusable mobile UI patterns rather than isolated screens.

Key principles:
- reusable screen shell
- consistent spacing and layout rhythm
- safe-area aware composition
- scalable UI structure for future product growth
- product-focused interaction clarity
